JS Jargon & Review

General thingies:

  • {} - curlies, braces, curly braces
  • [] - brackets, square braces
  • () - parens
  • . - dot
  • # - pound, hash
  • ! - bang
  • $ - bling
  • / - whack
  • * - splat
  • " " - quotes
  • ' ' - single quotes

Proper Nouns:

  • variables - something to which we can assign a value (eg. string, number)
  • functions - repeatable stack of code, declared as expressions (within a variable), can get args which are declared as params, may return values (if wanted)
  • objects - something can store data (variable types, key value pairs) and behavior (functions/methods) about itself
  • arrays - list of things by index
